Hướng dẫn tạo web tĩnh - trang cá nhân trên Github

2019-02-28 Nguyễn Hồng Linh
Nói đến trang web các nhân, mọi người hay nghĩ đến các trang blog cá nhân, và nghĩ đến các Opensource CMS (Content Managenment System, hệ thống quản lý và xuất bản bài viết có mã nguồn mở) CMS thường sẽ gồm 2 phần, phần Admin để bạn quản lý cơ sở dữ liệu backend (các bài viết, comment, media, ảnh…) và phần frontend là trang clent (trang chủ) được publish (đăng lên) các nội dung bài viết từ admin cho người dùng đọc.
